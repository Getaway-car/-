## SpringBoot+vue前后端分离的环境监测系统
本系统通过扫描本地文件，读取数据到数据库，并进行数据展示  
文件路径在EnvironmentServiceImpl下的scanTxt方法中修改
## 实用技术
后端：SpringBoot2.6.7，MybatisPlus  
前端：Vue2，element-ui，echarts5.0
